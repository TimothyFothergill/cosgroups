# Branding Decisions

## Colour Identity

We use the following hexadecimal values:

#6e026f - Purple
#abdadc - Light blue
#f1e6c9 - Beige
#fa891a - Orange

- Big, in your face parts of a page should be #6e026f
- General page stuff should be #abdadc
- Background should be #f1e6c9
- Call To Actions should be #fa891a
